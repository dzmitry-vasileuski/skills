# Agent Skills

Collection of skills for AI agents.

## Available Skills

| Skill | Description |
|-------|-------------|
| [excalidraw](./skills/excalidraw) | Generate .excalidraw diagram files — flowcharts, architecture diagrams, ER diagrams, sequence diagrams, mind maps |
| [grepai](./skills/grepai) | Semantic code search with grepai CLI — finds code by meaning, not text |
| [laravel-docs](./skills/laravel-docs) | Search official Laravel ecosystem documentation using semantic vector search |

## Installation

```bash
npx skills add dzmitry-vasileuski/skills --skill laravel-docs
```
